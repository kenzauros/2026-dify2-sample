### 第2回 作ったワークフローをツール化 ― チャットフローでの活用

- ワークフロー [Webページ要約.yml](./02/Dify-DSL/Webページ要約.yml)
- チャットフロー [Webページ要約ボット.yml](./02/Dify-DSL/Webページ要約ボット.yml)


#### インポート手順

チャットフローから利用しているワークフローのIDが環境によって異なるため、DSLインポート時は以下の手順でインポートしてください。
1. `Webページ要約.yml` をDifyにインポートして公開し、さらにツールとして公開します（手順は記事を参照）
2. `Webページ要約ボット.yml` をDifyにインポートします
3. チャットフローの「Webページ要約」ノードを右クリックして「ノード変更」→「ツール」タブ→「Webページ要約」を選択
4. 新しい「Webページ要約」ノードをユーザー入力、テンプレートノードと接続
5. チャットフローを公開

※Webページ要約ワークフローは第2回と共通です。

#### インポート手順

チャットフローから利用しているワークフローのIDが環境によって異なるため、DSLインポート時は以下の手順でインポートしてください。
1. `Webページ要約.yml` をDifyにインポートして公開し、さらにツールとして公開します（手順は記事を参照）
2. `Webページ要約ボットv2.yml` をDifyにインポートします
3. チャットフローの「Webページ要約」ノードを右クリックして「ノード変更」→「ツール」タブ→「Webページ要約」を選択
4. 新しい「Webページ要約」ノードをイテレーションの開始ノードと接続
5. チャットフローを公開
